# ShareAny

用于共享一些文件.

## 开发计划

* [x] 上传功能
* [x] 文本
* [x] 图片
* [x] 音乐
* [x] 文件
* [x] 视频
* [x] 点击文本复制
* [x] HTTPS
* [x] 随机链接防偷窥
* [x] 自由更改端口
* [x] 系统托盘常驻
* [ ] 自定义快捷键（默认 Ctrl_Space 从剪贴板添加 Ctrl_D清空)
* [ ] 组 （一次粘贴的文件在一个控件中显示）
* [ ] 文件类型图标 (文件类型显示时添加图标)

---

* [ ] html 从文件建立页面
