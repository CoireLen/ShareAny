# ShareAny

用于共享一些文件.

## 开发计划

* [X] 上传功能
* [X] 文本
* [X] 图片
* [X] 音乐
* [X] 文件
* [X] 视频
* [X] 点击文本复制
* [X] HTTPS
* [X] 随机链接防偷窥
* [X] 自由更改端口
* [ ] 系统托盘常驻
* [ ] 自定义快捷键（默认 Ctrl_Space 从剪贴板添加 Ctrl_D清空)
* [ ] 组 （一次粘贴的文件在一个控件中显示）
* [ ] 文件类型图标 (文件类型显示时添加图标)

---

* [ ] html 从文件建立页面
